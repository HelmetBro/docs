title: "Quickstart"
description: "Send your first message and receive webhooks in minutes."
---

<Steps>
<Step title="Get your API key">
  Sign in to your Blooio account and create an API key.
  
  <Warning>Keep your API key secret. Never commit it to source control.</Warning>
</Step>

<Step title="Send your first message">
  Use the Messages endpoint to queue a message.
  
  <RequestExample>
  ```bash cURL
  curl -X POST 'https://edge.blooio.com/v1/api/messages' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -H 'Content-Type: application/json' \
    -d '{
      "to": "+15551234567",
      "text": "Hello from Blooio!"
    }'
  ```
  
  ```javascript Node.js
  import fetch from 'node-fetch'
  
  const res = await fetch('https://edge.blooio.com/v1/api/messages', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.BLOOIO_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ to: '+15551234567', text: 'Hello from Blooio!' })
  })
  const data = await res.json()
  ```
  </RequestExample>
  
  <Check>Response includes `message_id` and status `queued`.</Check>
</Step>

<Step title="Receive webhook events">
  Configure your webhook URL to receive delivery updates and inbound messages.
  
  <RequestExample>
  ```bash cURL
  curl -X PUT 'https://edge.blooio.com/v1/api/config/webhook' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -H 'Content-Type: application/json' \
    -d '{ "webhook_url": "https://example.com/mywebhook" }'
  ```
  </RequestExample>
  
  <Info>See all webhook payloads in <a href="/api-reference/webhooks/events">Webhook events</a>.</Info>
</Step>

<Step title="Track message status">
  Poll for status or retrieve full details.
  
  ```bash
  curl -H 'Authorization: Bearer YOUR_API_KEY' \
    'https://edge.blooio.com/v1/api/messages/{messageId}/status'
  ```
</Step>
</Steps>
