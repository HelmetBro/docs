---
title: "Get message status"
description: "Retrieve only the current delivery status of a message by ID."
---

## GET /v1/api/messages/{messageId}/status

### Path parameters

<ParamField path="messageId" type="string" required>
Unique identifier for the message.
</ParamField>

### Headers

<ParamField header="Authorization" type="string" required>
Bearer token in the format `Bearer YOUR_API_KEY`.
</ParamField>

### Example request

<RequestExample>
```bash cURL
curl -X GET 'https://edge.blooio.com/v1/api/messages/gK2Ig_XGR2M6UkSgmT9FK/status' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

```javascript Node.js
import fetch from 'node-fetch';

const res = await fetch('https://edge.blooio.com/v1/api/messages/gK2Ig_XGR2M6UkSgmT9FK/status', {
  headers: { 'Authorization': `Bearer ${process.env.BLOOIO_API_KEY}` }
});
const data = await res.json();
```
</RequestExample>

### Response

<ResponseExample>
```json 200 OK
{
  "message_id": "gK2Ig_XGR2M6UkSgmT9FK",
  "status": "delivered"
}
```
</ResponseExample>

#### Error responses

<ResponseExample>
```json 401 Unauthorized
{ "error": "unauthorized", "message": "Invalid or missing API key", "status": 401 }
```
</ResponseExample>

<ResponseExample>
```json 404 Not Found
{ "error": "not_found", "message": "Message not found", "status": 404 }
```
</ResponseExample>
