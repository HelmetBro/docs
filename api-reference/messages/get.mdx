---
title: "Get message details"
description: "Retrieve metadata and the current status of a message by ID."
---

## GET /v1/api/messages/{messageId}

### Path parameters

<ParamField path="messageId" type="string" required>
Unique identifier for the message.
</ParamField>

### Headers

<ParamField header="Authorization" type="string" required>
Bearer token in the format `Bearer YOUR_API_KEY`.
</ParamField>

### Example request

<RequestExample>
```bash cURL
curl -X GET 'https://edge.blooio.com/v1/api/messages/gK2Ig_XGR2M6UkSgmT9FK' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

```javascript Node.js
import fetch from 'node-fetch';

const res = await fetch('https://edge.blooio.com/v1/api/messages/gK2Ig_XGR2M6UkSgmT9FK', {
  headers: { 'Authorization': `Bearer ${process.env.BLOOIO_API_KEY}` }
});
const data = await res.json();
```
</RequestExample>
