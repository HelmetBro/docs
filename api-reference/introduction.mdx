## Overview

The Blooio Messaging API lets you send and receive messages over iMessage and SMS. All endpoints are prefixed with `/v1/api` and secured with a Bearer API key.

## Authentication

Include your API key in the `Authorization` header.

```bash
Authorization: Bearer YOUR_API_KEY
```

Base URL:

```bash
https://edge.blooio.com
```
