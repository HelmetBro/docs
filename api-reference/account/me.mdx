## GET /v1/api/me

### Headers

<ParamField header="Authorization" type="string" required>
Bearer token in the format `Bearer YOUR_API_KEY`.
</ParamField>

### Example request

<RequestExample>
```bash cURL
curl -X GET 'https://edge.blooio.com/v1/api/me' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

```javascript Node.js
import fetch from 'node-fetch';

const res = await fetch('https://edge.blooio.com/v1/api/me', {
  headers: { 'Authorization': `Bearer ${process.env.BLOOIO_API_KEY}` }
});
const data = await res.json();
```

```python Python
import os, requests

res = requests.get('https://edge.blooio.com/v1/api/me',
  headers={ 'Authorization': f"Bearer {os.environ['BLOOIO_API_KEY']}" }
)
data = res.json()
```
</RequestExample>
